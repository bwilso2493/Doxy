Detailed Description

This class is a wrapper class for Game Objects (mainly 3D) that need to have certain abilities to have access to different features such as collisions, alarms, inputs, etc.

If you want a class to be a Game Object just derive that class from GameObject.

◆ SetCurrentCamera()

void GameObject::SetCurrentCamera ( Camera *  pCam)

This function sets the current camera of the user created scene.

Example This is an example of the user creating a camera and setting the camera to be current.

void SceneDemo::Initialize()
{
// === create a new Camera === //
pCamera = new Camera(Camera::Type::PERSPECTIVE_3D);
// === Set the necessary perspective and viewports === //
pCamera->setViewport(0, 0, Chaos::GetWidth(), Chaos::GetHeight());
pCamera->setPerspective(35.0f, float(Chaos::GetWidth()) / float(Chaos::GetHeight()), 1.0f, 5000.0f);
// === Orient Camera === //
Vect up3DCam(0.0f, 1.0f, 0.0f);
Vect pos3DCam(50.0f, 50.0f, 150.0f);
Vect lookAt3DCam(0.0f, 0.0f, 0.0f);
pCamera->setOrientAndPosition(up3DCam, lookAt3DCam, pos3DCam);
pCamera->updateCamera(); // Note: if the camera moves/changes, we need to call update again
// === set the current camera === //
SetCurrentCamera(pCamera);
}

pCam The Camera that the user would like to be the current camera in the scene.

◆ SubmitEntry()

void GameObject::SubmitEntry ( )

This function will submit entry to scene meaning that the game object will call virtual void SceneEntry() the next frame.

Example This is an example of using the SubmitEntry() function in a bullet factory.

PlayerBullet* BulletFactory::privCreatePlayerBullet(Vect M, Matrix& mat, Matrix& rot)
{
PlayerBullet* b;
if (ActivePlayerBullets.empty())
{
b = new PlayerBullet();
ActivePlayerBullets.push_front(b);
// DebugMsg::out("New Bullet Created\n");
}
else
{
b = InactivePlayerBullets.back();
InactivePlayerBullets.pop_front();
DebugMsg::out("Recycled Bullet used\n");
}
b->Initialize(M, mat, rot);
b->SubmitEntry();
return b;
}
