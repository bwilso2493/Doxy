Detailed Description

A manager that manages the pre loaded shaders the user wants from the LoadAllResources() file. Utiliizes a map so that the user can create a key to their personal liking. Also allows very fast access to get the shader a user might be looking for.

◆ GetShader()

static ShaderObject * ShaderManager::GetShader ( const std::string  shaderKey)
inlinestatic

This function will return the shader that the user wants. The shader returned is specified by the user using the shaderKey the user used when loading the shader in the LoadAllResources() file.

Note
The spelling of the shaderKey matters. This includes lowercase and uppercase letters.

Example This is an example of how this function is used to get the shader for a tank body.

PlayerTank::PlayerTank()
{
pBody = new GraphicsObject_TextureLight(ModelManager::GetModel("TankBody"), ShaderManager::GetShader("TextureLight"), TextureManager::GetTexture("Body"), TextureManager::GetTexture("Body"), TextureManager::GetTexture("Track"), LightColor, LightPos);
}
Parameters
shaderKeyThe shader key is the identifier the user created.
Returns
If the shaderKey is found, than the shader is returned. If the shader key is not found, than

than the function will throw an assert.

inlinestatic

This function is used by the user in LoadAllResources() and preloads the shader the user wants into the game. This is done by inserting into a map.

Example This is an example of the user loading a shader within the LoadAllResources() file.

void Chaos::LoadAllResources()
{
ShaderManager::LoadShader("sprite", "spriteRender");
}
