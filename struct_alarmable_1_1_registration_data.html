Detailed Description

A struct that will maintain each alarms data.

Note
float alarmTimer is used for the amount of time that will pass before alarm is called 
   AlarmableManager::StorageEntryRef pMyDeleteRef is reference to the Timeline multimap
   within AlarmManager.  This allows for faster deletion from the multimap.

   RegistrationState regState is an enum created to ensure that no alarm is registered or
   dergistered multiple times.

       AlarmRegistrationCommand* alarmReg is a pointer to command base to register the alarm.

       AlarmDeregistrationCommand* alarmDereg is a pointer to command base to deregister the alarm.
