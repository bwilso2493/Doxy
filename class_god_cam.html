Detailed Description

This class creates a GodCam for the user. The GodCam allows the user to have control over the camera which could act as a debugging tool for the user.

Example This is an example of how to create a GodCam.

void Plane::CreateGodCam()
{
GodCam* pGodCam = new GodCam();
pGodCam->CreateGodCam();
pGodCam->GodCamBecomesCurrentCam();
}
