Each GameObjects can have set up to 5 seperate alarms (numbered 0 through 4) and have the associated callback triggered.

Alarms are simple countdown mechanism to help control time-based events. Below is a simple example of a self-resetting alarm to make the Bug move about randomly. (Please refer to sf::Shape for more information about its member functions)

Bug::Bug( )
{
myshape.setPointCount(3);
myshape.setRadius( 15 );
myshape.setPosition( 100, 100 );
SetAlarm( 1, 2); // sets Alarm 1 to go off in 2 secs
}
void Bug::Update()
{
// ... move myshape forward...
}
void Bug::Alarm1() // This is the Alarm1 callback
{
myshape.rotate( (float) (90 - 2 * ( rand() % 90 )) ); // adds a random rotation angle [-45..45] degrees
SetAlarm( 1, 1.0f + rand() % 4); // Resets Alarm 1 to a random value [1..5] secs;
}
