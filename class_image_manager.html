Detailed Description

A manager that manages the pre loaded images the user wants from the LoadAllResources() file. Utiliizes a map so that the user can create a key to their personal liking. Also allows very fast access to get the image a user might be looking for.

◆ GetImage()

static Image * ImageManager::GetImage ( const std::string  imageKey)
inlinestatic

This function will return the image that the user wants. The image returned is specified by the user using the imageKey the user used when loading the image in the LoadAllResources() file.

Note
The spelling of the imageKey matters. This includes lowercase and uppercase letters.

Example This is an example of how this function is used when creating a new Sprite.

PlayerTank::PlayerTank()
{
pSprite = new Sprite(ImageManager::GetImage("Stitch"), new Rect(100.0f, 100.0f, 100.0f, 100.0f));
}
Parameters
imageKeyThe image key is the identifer that the user set in load image within LoadAllResources.
Returns
If the imageKey is found, than the image is returned. If the image key is not found, than

than the function will throw an assert.
