Chaos Engine

Chaos is an engine that allows simple games to be created in either 2D or 3D.

Chaos Features

Chaos contains these features:

  • A resource manager organize your external assets such as textures, images, etc.
  • Base classes Scene and GameObject from which to derive your own levels and game entities.
  • A built-in game loop to update and draw the current Scene and its associated GameObject.
  • A built-in collision system for GameObject which tests collisions based on Bounding Spheres of GameObjects.
  • A built-in alarm system which allows users to use up to 3 alarms per game object.
  • A built-in input management system that allows the user to use as many keys as they wish.
  • A built-in time management system that keeps track of time and also contains a freeze frame tool.
