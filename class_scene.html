Detailed Description

This class is used by the user as a base class for the scene the user wishes to create. The scene class allows the user to not have to worry about including certain aspects within

their own code such as an update loop or a draw function.

void Scene::Initialize ( )
privatepure virtual

This is a virtual function that the initializes everything the user wants within their own scene.

Note
This function MUST be overriden for every scene created.

Example This is an example of a scene being initialized and creating a plane and player tank object.

void SceneDemo2::Initialize()
{
GOPlane = new Plane(Matrix(SCALE, 2000.0f, 2000.0f, 2000.f));
pPlayer = new PlayerTank();
}

privatepure virtual

This is a virtual function that will be called when the user either closes the game or changes to a new scene.

Note
This function MUST be overriden for every scene created.
Example This is an example of a the SceneEnd function be called and deregistering the plane and player tank from the current scene.
void SceneDemo2::SceneEnd()
{
GOPlane->Deregister();
pPlayer->Deregister();
}
