Detailed Description

A manager that manages the pre loaded models the user wants from the LoadAllResources() file. Utiliizes a map so that the user can create a key to their personal liking. Also allows very fast access to get the model a user might be looking for.

◆ GetModel()

static Model * ModelManager::GetModel ( const std::string  modelKey)
inlinestatic

This function will return the model that the user wants. The model returned is specified by the user using the modelKey the user used when loading the model in the LoadAllResources() file.

Note
The spelling of the modelKey matters. This includes lowercase and uppercase letters.

Example This is an example of how this function is used to get the model for a tank body.

PlayerTank::PlayerTank()
{
pBody = new GraphicsObject_TextureLight(ModelManager::GetModel("TankBody"), ShaderManager::GetShader("TextureLight"), TextureManager::GetTexture("Body"), TextureManager::GetTexture("Body"), TextureManager::GetTexture("Track"), LightColor, LightPos);
}
Parameters
modelKeyThe model key identifer that the user created.
Returns
If the modelKey is found, than the model is returned. If the model key is not found, than

than the function will throw an assert.

This function is used by the user in LoadAllResources() and preloads the model the user wants into the game. This is done by inserting into a map.

Example This example shows how the user would load a Model within the LoadAllResources file.

void Chaos::LoadAllResources()
{
ModelManager::LoadModel("Tank", "tank.azul");
}
