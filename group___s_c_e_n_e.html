A Scene is the space in which GameObjects exist.

Scenes are the sandbox in which GameObjects exists. Each level of a game will have a corresponding Scene of course, but other important game steps such as splash pages, menu selection, inventory, etc. will likely use a seperate Scene.

When deriving from Scene, there is only one method to implement: Scene::Initialize. This method is called when creating a new Scene. Using this method, create all the initial GameObject that are required as well as indicating which collision pair are expected to be used.
