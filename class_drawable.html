Detailed Description

This class is used for GameObjects to allow them to be drawn on the screen. The class is setup to allow the user to only have to SubmitDrawRegistration one time and the Scene base class

will take care of actually drawing that object.

This function registers the object into the scene command which will allow the object to be drawn.

Example This is an example of an Axis object being submitted to be drawn.

Axis::Axis()

SubmitDrawRegistration();
}

What is actually drawn is up to the user based upon the virtual void Draw() method.
