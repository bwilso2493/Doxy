Detailed Description

A manager that manages the pre loaded texture the user wants from the LoadAllResources() file. Utiliizes a map so that the user can create a key to their personal liking. Also allows very fast access to get the texture a user might be looking for.

◆ GetTexture()

static Texture * TextureManager::GetTexture ( const std::string  textureKey)
inlinestatic

This function will return the texture that the user wants. The texture returned is specified by the user using the textureKey the user used when loading the texture in the LoadAllResources() file.

Note
The spelling of the textureKey matters. This includes lowercase and uppercase letters.

Example This is an example of how this function is used to get the texture for a tank body.

PlayerTank::PlayerTank()
{
pBody = new GraphicsObject_TextureLight(ModelManager::GetModel("TankBody"), ShaderManager::GetShader("TextureLight"), TextureManager::GetTexture("Body"), TextureManager::GetTexture("Body"), TextureManager::GetTexture("Track"), LightColor, LightPos);
}
Parameters
textureKeyThe texture key is the identifier the user created.
Returns
If the textureKey is found, than the texture is returned. If the texture key is not found, than

than the function will throw an assert.

This function is used by the user in LoadAllResources() and preloads the texture the user wants into the game. This is done by inserting into a map.

Example This is an example of the user loading a texture within the LoadAllResources() file.

void Chaos::LoadAllResources()
{
TextureManager::LoadTexture("Stitch", "stitch.tga");
}
