Detailed Description

This class is used for GameObjects to allow them to be updated within the game. The class is setup to allow the user to only have to SubmitUpdateRegistration one time and the Scene base class will take care of actually updating that object.

void Updateable::SubmitUpdateDeregistration ( )
protected

This function deregisters the object into the scene command which will no longer allow the object to be updated by the engine.

Example This is an example of submitting an update deregistration to stop the PlayerTank object from being updated in the scene.

PlayerTank::PlayerTank()
{
SubmitUpdateDeregistration();
}

◆ SubmitUpdateRegistration()

void Updateable::SubmitUpdateRegistration ( )
protected

This function registers the object into the scene command which will allow the object to be updated by the engine.

Example This is an example of an PlayerTank object being submitted to be updated.

PlayerTank::PlayerTank()
{
SubmitUpdateRegistration();
}
Note
What is actually updated is up to the user based upon the virtual void Update() method.

Example 2 This is an example of where the PlayerTank Update is actually called.

void PlayerTank::Update()
{
// Tank translation movement (not using time-based values for simplicity)
if (Keyboard::GetKeyState(AZUL_KEY::KEY_W))
{
tankBodyPos += Vect(0, 0, 1) * tankRot * tankSpeed;
tankTurretPos += Vect(0, 0, 1) * tankRot * tankSpeed;
}
if (Keyboard::GetKeyState(AZUL_KEY::KEY_S))
{
tankBodyPos += Vect(0, 0, 1) * tankRot * -tankSpeed;
tankTurretPos += Vect(0, 0, 1) * tankRot * -tankSpeed;
}
// Tank Rotation movement (not using time-based values for simplicity)
if (Keyboard::GetKeyState(AZUL_KEY::KEY_A))
{
tankRot *= Matrix(ROT_Y, tankRotAng);
}
else if (Keyboard::GetKeyState(AZUL_KEY::KEY_D))
{
tankRot *= Matrix(ROT_Y, -tankRotAng);
}
}
